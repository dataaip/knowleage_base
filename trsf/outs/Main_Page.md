# C 和 C++ 参考手册

## C++ 参考手册

### 版本及编译器支持
- **标准版本**: C++11, C++14, C++17, C++20, C++23, C++26
- **编译器支持**: C++11, C++14, C++17, C++20, C++23, C++26

### 语言特性
#### 关键字
C++ 语言的关键字是预定义标识符，具有特定含义，不能作为变量名使用。例如:
```cpp
int class = 10; // 错误，class 是关键字
```

#### 预处理器
预处理器在编译之前处理源代码中的指令。常见的预处理器指令包括 `#include`, `#define`, `#ifdef` 等。例如：
```cpp
#include <iostream>
#define PI 3.14159

int main() {
    std::cout << "The value of PI is " << PI << std::endl;
    return 0;
}
```

#### 基本概念
- **注释**: 注释分为单行注释 (`//`) 和多行注释 (`/* ... */`)。
- **名称查找**: C++ 支持作用域规则，如全局作用域、局部作用域等。
- **类型**: 包括基本类型（如 `int`, `float`）、复合类型（如数组、指针）和用户自定义类型（如类、结构体）。
- **`main` 函数**: 程序入口点，返回 `int` 类型。
- **表达式**: 包含操作数和运算符，如 `a + b`。
- **值类别**: 包括左值、右值、纯右值、将亡值等。
- **求值顺序**: 表达式的求值顺序在 C++ 中通常是未定义的，除了某些明确规定的顺序。
- **运算符**: 包括算术运算符、逻辑运算符、位运算符等。
- **转换**: 包括隐式转换（如自动类型转换）和显式转换（如 `static_cast`）。
- **语句**: 包括简单语句、复合语句、条件语句、循环语句等。
- **声明和初始化**: 声明引入一个名称到作用域中，初始化赋予变量初始值。
- **函数**: 可重载，允许多个同名但参数不同的函数。
- **类和联合体**: 类支持继承、封装、多态；联合体共享内存。
- **模板**: 允许编写泛型代码。
- **异常**: 使用 `try-catch` 块捕获异常。
- **独立实现**: 提供了标准库实现的替代方案。
- **标准库**: 提供丰富的库支持，如 STL。

### 标准库
#### 语言支持库
- **程序工具**: 包括信号处理、非局部跳转、基本内存管理、可变参数函数等。
- **诊断库**: 包括断言、系统错误、异常类型、错误码等。
- **内存管理库**: 包括分配器、智能指针、内存资源等。
- **元编程库**: 包括类型特征、ratio、integer_sequence 等。
- **通用工具库**: 包括函数对象、hash、交换、类型操作、整数比较、pair、tuple、optional、expected、variant、any、bitset 等。
- **容器库**: 包括 vector、deque、array、list、forward_list、map、multimap、set、multiset、unordered_map、unordered_multimap、unordered_set、unordered_multiset、容器适配器、span、mdspan 等。
- **迭代器库**: 提供遍历容器元素的机制。
- **范围库**: 提供更简洁的遍历方式（C++20 引入）。
- **算法库**: 包括排序、搜索、复制等算法。
- **字符串库**: 包括 basic_string、char_traits、basic_string_view 等。
- **文本处理库**: 包括原始数值转换、格式化、locale、字符分类、text_encoding、正则表达式等。
- **数值库**: 包括数学函数、数学特殊函数、数学常量、基本线性代数算法、数据并行类型、伪随机数生成、浮点环境、complex、valarray 等。
- **日期和时间库**: 包括日历、时区等。
- **输入输出库**: 包括打印函数、流式 I/O、basic_istream、basic_ostream、同步输出、文件系统等。
- **并发支持库**: 包括 thread、jthread、atomic、互斥、条件变量、latch、barrier、安全回收等。
- **执行支持库**: 提供对程序执行的支持（C++26 引入）。

### 技术规范
#### 标准库扩展
- **标准库扩展 (基础库 TS)**: 包括 resource_adaptor、invocation_type 等。
- **标准库扩展 v2 (基础库 TS v2)**: 包括 propagate_const、ostream_joiner、randint、observer_ptr、检测惯用法 等。
- **标准库扩展 v3 (基础库 TS v3)**: 包括 scope_exit、scope_fail、scope_success、unique_resource 等。
- **并行性库扩展 v2 (并行性 TS v2)**: 包括 simd 等。
- **并发库扩展 (并发 TS)**: 提供额外的并发支持。
- **事务内存 (TM TS)**: 提供事务内存支持。
- **反射 (反射 TS)**: 提供运行时类型信息和反射能力。

## C 参考手册

### 版本及编译器支持
- **标准版本**: C89, C95, C99, C11, C17, C23
- **编译器支持**: C99, C23

### 语言特性
#### 基本概念
- **关键字**: 包括 `auto`, `break`, `case`, `char`, `const`, `continue`, `default`, `do`, `double`, `else`, `enum`, `extern`, `float`, `for`, `goto`, `if`, `inline`, `int`, `long`, `register`, `return`, `short`, `signed`, `sizeof`, `static`, `struct`, `switch`, `typedef`, `union`, `unsigned`, `void`, `volatile`, `while`。
- **预处理器**: 同 C++。
- **表达式**: 同 C++。
- **声明**: 包括变量声明、函数声明等。
- **初始化**: 同 C++。
- **函数**: 同 C++。
- **语句**: 同 C++。

### 标准库
#### 头文件
- 提供各种功能，如输入输出、字符串操作、数学计算等。

#### 类型支持
- 提供基本类型和一些辅助类型定义。

### 程序工具
- 同 C++。

### 可变参数函数
- 提供 `stdarg.h` 头文件来支持可变参数函数。

### 诊断库
- 同 C++。

### 动态内存管理
- 提供 `malloc`, `free`, `calloc`, `realloc` 等函数进行动态内存管理。

### 字符串库
- 提供 `string.h` 头文件中的字符串操作函数。

### 日期和时间库
- 提供 `time.h` 头文件中的日期和时间操作函数。

### 本地化库
- 提供 `locale.h` 头文件中的本地化支持。

### 输入输出库
- 提供 `stdio.h` 头文件中的输入输出函数。

### 算法库
- 提供 `stdlib.h` 头文件中的算法函数，如排序、查找等。

### 数值库
- 提供 `math.h` 头文件中的数学函数。
- 提供 `complex.h` 头文件中的复数算术函数（C99 引入）。
- 提供 `tgmath.h` 头文件中的类型通用数学函数（C99 引入）。
- 提供 `fenv.h` 头文件中的浮点环境函数（C99 引入）。
- 提供 `stdalign.h` 头文件中的对齐支持（C11 引入）。
- 提供 `stdatomic.h` 头文件中的原子操作支持（C11 引入）。
- 提供 `threads.h` 头文件中的线程支持（C11 引入）。
- 提供 `uchar.h` 头文件中的宽字符支持（C11 引入）。
- 提供 `wchar.h` 头文件中的宽字符支持（C95 引入）。
- 提供 `wctype.h` 头文件中的宽字符分类支持（C95 引入）。

### 并发支持库
- 提供 `threads.h` 头文件中的线程支持（C11 引入）。

### 技术规范
#### 动态内存扩展 (动态内存 TR)
- 提供额外的动态内存管理功能。

#### 浮点扩展，第1部分 (FP Ext 1 TS)
- 提供浮点数的额外功能。

#### 浮点扩展，第4部分 (FP Ext 4 TS)
- 提供浮点数的额外功能。
