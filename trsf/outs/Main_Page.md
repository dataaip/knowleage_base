## C++ 参考

### 关键知识点扩展

#### 关键字 - 预处理器
- **关键字**：C++ 提供了一系列关键字用于定义程序结构和行为，例如 `class`, `namespace`, `template` 等。
- **预处理器**：预处理器指令在编译之前处理代码，常见的有 `#include`, `#define`, `#ifdef` 等。例如：
  ```cpp
  #define PI 3.14159
  ```

#### ASCII 表
- ASCII（American Standard Code for Information Interchange）是美国信息交换标准代码，用于表示128种字符。例如，字符 'A' 对应的 ASCII 码是 65。

#### 基本概念
- **注释**：注释用于解释代码，分为单行注释（`//`）和多行注释（`/* ... */`）。
- **名称（查找）**：名称查找涉及标识符在不同作用域中的可见性和解析，遵循作用域规则。
- **类型（基础类型）**：C++ 支持多种基础类型，如 `int`, `float`, `double`, `char` 等。例如：
  ```cpp
  int a = 5;
  float b = 3.14f;
  ```
- **`main` 函数**：程序的入口点，返回类型为 `int`。例如：
  ```cpp
  int main() {
      return 0;
  }
  ```

#### 表达式
- **值类别**：表达式可以分为左值（lvalue）、右值（rvalue）、纯右值（prvalue）、将亡值（xvalue）。
- **求值顺序**：C++ 标准规定了一些表达式的求值顺序，如短路求值。
- **运算符（优先级）**：C++ 中有丰富的运算符，包括算术运算符、关系运算符、逻辑运算符等。例如：
  ```cpp
  int result = (a + b) * c;
  ```
- **转换**：C++ 支持隐式和显式类型转换，例如：
  ```cpp
  double d = static_cast<double>(a) / b;
  ```
- **字面量**：字面量是直接出现在代码中的固定值，例如整数字面量、浮点数字面量、字符字面量等。

#### 语句
- **`if` - `switch`**：条件语句用于根据条件执行不同的代码块。例如：
  ```cpp
  if (a > b) {
      // do something
  } else {
      // do something else
  }

  switch (choice) {
      case 1:
          // do something
          break;
      case 2:
          // do something else
          break;
      default:
          // default action
  }
  ```
- **`for` - 范围`for` (C++11)**：循环语句用于重复执行代码块。例如：
  ```cpp
  for (int i = 0; i < n; ++i) {
      // do something
  }

  for (auto x : vec) {
      // do something with x
  }
  ```
- **`while` - `do`-`while`**：循环语句根据条件重复执行代码块。例如：
  ```cpp
  while (condition) {
      // do something
  }

  do {
      // do something
  } while (condition);
  ```

#### 声明 - 初始化
- **声明**：声明引入新的名称到程序中，指定其类型。例如：
  ```cpp
  int a;
  ```
- **初始化**：初始化是指定新对象初始值的过程。例如：
  ```cpp
  int a = 5;
  ```

#### 函数 - 重载
- **函数**：函数是一段完成特定任务的代码块。例如：
  ```cpp
  void print(int a) {
      std::cout << a;
  }
  ```
- **重载**：允许同一个名称有多个实现，通过参数列表区分。例如：
  ```cpp
  void print(int a) {
      std::cout << a;
  }

  void print(double a) {
      std::cout << a;
  }
  ```

#### 类（联合体）
- **类**：类是面向对象编程的基本构建块，包含数据成员和成员函数。例如：
  ```cpp
  class Point {
  public:
      int x, y;
      void set(int a, int b) {
          x = a;
          y = b;
      }
  };
  ```
- **联合体**：联合体是一种特殊的数据结构，所有成员共享同一块内存。例如：
  ```cpp
  union Data {
      int i;
      float f;
      char str[20];
  };
  ```

#### 模板 - 异常
- **模板**：模板用于创建泛型编程结构，提高代码复用性。例如：
  ```cpp
  template<typename T>
  T max(T a, T b) {
      return (a > b) ? a : b;
  }
  ```
- **异常**：异常处理机制用于处理程序运行时错误。例如：
  ```cpp
  try {
      throw std::runtime_error("Error occurred");
  } catch (const std::exception& e) {
      std::cerr << e.what();
  }
  ```

### 关键知识点扩展

#### 基本概念
- **类型支持**：C 语言支持多种数据类型，如 `int`, `float`, `double`, `char` 等。
- **程序工具**：提供一些辅助程序功能的工具，如 `sizeof` 运算符。
- **表达式**：表达式是由操作数和运算符组成的，计算结果为一个值。
- **声明**：声明引入新的名称到程序中，指定其类型。
- **初始化**：初始化是指定新对象初始值的过程。
- **函数**：函数是一段完成特定任务的代码块。
- **语句**：语句是程序的基本执行单元，包括控制流程语句。

#### 关键字
- C 语言的关键字包括 `int`, `float`, `double`, `char`, `if`, `else`, `for`, `while` 等。

#### 预处理器
- **预处理器指令**：预处理器指令在编译之前处理代码，常见的有 `#include`, `#define`, `#ifdef` 等。

#### 头文件
- **类型支持**：包含各种类型定义的头文件，如 `<stddef.h>`。
- **程序工具**：包含一些辅助程序功能的头文件，如 `<stdio.h>`。
- **诊断库**：包含错误处理相关的头文件，如 `<assert.h>`。
- **动态内存管理**：包含动态内存管理相关的头文件，如 `<stdlib.h>`。
- **字符串库**：包含字符串处理相关的头文件，如 `<string.h>`。
- **日期和时间库**：包含日期和时间处理相关的头文件，如 `<time.h>`。
- **本地化库**：包含本地化支持相关的头文件，如 `<locale.h>`。
- **输入/输出库**：包含输入输出相关的头文件，如 `<stdio.h>`。

#### 算法库
- **数值库**：包含数学函数和常量的头文件，如 `<math.h>`。
- **浮点环境** (C99)：包含浮点环境控制的头文件，如 `<fenv.h>`。
- **伪随机数生成**：包含伪随机数生成函数的头文件，如 `<stdlib.h>`。
- **复数运算** (C99)：包含复数运算的头文件，如 `<complex.h>`。
- **泛型数学** (C99)：包含泛型数学函数的头文件，如 `<tgmath.h>`。
- **位操作** (C23)：包含位操作函数的头文件，如 `<bitops.h>`。
- **检查整数算术** (C23)：包含整数算术检查函数的头文件，如 `<stdckdint.h>`。
- **并发支持库** (C11)：包含线程支持的头文件，如 `<threads.h>`。
